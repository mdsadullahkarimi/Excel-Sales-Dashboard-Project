# 📊 Superstore Sales Analysis Dashboard (Excel)

## 📌 Project Overview

This project analyzes Superstore sales data using Microsoft Excel. The dashboard provides interactive insights into sales performance across categories, regions, customer segments, and top customers.

---

## 🎯 Objectives

- Analyze overall sales performance
- Identify top-performing product categories
- Compare sales across regions
- Track monthly sales trends
- Find top customers
- Build an interactive dashboard using Excel

---

## 🛠 Tools Used

- Microsoft Excel
- Power Query
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas

---

## 📊 Dashboard Features

- ✅ Total Sales KPI
- ✅ Total Orders KPI
- ✅ Total Customers KPI
- ✅ Average Sales per Order
- ✅ Sales by Category
- ✅ Monthly Sales Trend
- ✅ Sales by Region
- ✅ Sales by Segment
- ✅ Top 10 Customers

---

## 📂 Dataset

- Source: Kaggle Superstore Sales Dataset
